# 암호화 복호화 

<h1>jasypt의 PBEWithMD5AndDES 알고리즘을 사용 하여 key값을 입력 받아 암호화 및 복호화</h1>
sandom salt를 사용하는 경우는 암호화된 결과 값이 매번 바뀌기 때문에 StringFixedSaltGenerator를 사용 하여 테스트 하기 위해 암호화된 결과 값을 고정해서 사용하였음
